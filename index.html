How to create github web page?

1.     
Create a GitHub account at github.com. If
you already have an account, skip to step 2.


2.     
Create a new repository on GitHub by clicking
on the "+" sign on the top right corner and selecting "New
repository". Give it a name that's the same as your username, followed by â€œ.
github.io". For example, if my username is "AmnaMAwan", my
repository name " AmnaMAwan.github.io".


3.     
Once the repository is created, click on
the "Settings" tab, scroll down to the "GitHub Pages"
section, and select "Master branch" under the "Source"
dropdown menu.


4.     
Choose a theme for your website by
clicking on the "Change theme" button in the "GitHub Pages"
section. Select a theme you like, or you can add a custom theme if you prefer.


5.     
Now it's time to add content to your
website. You can create an "index.html" file in the repository, which
will be the main page of your website. You can use HTML, CSS, and JavaScript to
create your website.


6.     
Commit the changes you made to the
repository by clicking on the "Commit changes" button.
